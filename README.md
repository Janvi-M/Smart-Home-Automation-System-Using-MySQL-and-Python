This project implements a Smart Home Automation System utilizing Python for the application logic and MySQL for database management. 
The system allows users to manage various smart home devices, log maintenance activities, and view detailed logs of device operations.

Key Features:
1. Device Management: Add, update, and remove devices from the smart home setup.
2. Maintenance Logging: Automatically log maintenance tasks, including issues reported and next maintenance dates.
3. Log Display: View maintenance logs directly within the application.
4. User Notifications: Get alerts for maintenance activities and device status changes.
5. The integration of Python and MySQL provides a robust solution for efficient home management.
